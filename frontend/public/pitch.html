05 / How It Works

Architecture — Built to Scale

A production-grade RAG pipeline with agentic tool-use, deployed on serverless infrastructure. Every layer is independently scalable.

📄 Policy PDFs

Customer uploads via web UI or local Google Drive sync. Any insurer, any format.

⚙️ Ingestion Pipeline

PDF extraction → semantic chunking → OpenAI embeddings (text-embedding-3-small). SHA-256 dedup prevents re-processing.

🗄️ Supabase Vector DB

pgvector with HNSW index. JSONB metadata per chunk: policy type, provider, premium, renewal date, insured entity.

↓
🔧 Broker MCP Server

Railway-hosted. Tools: semantic search, policy inventory, renewal calendar. Also handles PDF upload and metadata updates.

💲 Quote MCP Server

Stateless Railway service. Tools: home / motor / pet quotes from 3 insurers, GPT-4o-mini photo analysis for asset identification.

↓
🤖 Claude (claude-sonnet-4-6) — Agentic Tool-Use Loop

Orchestrates multi-step reasoning across both MCP servers. Searches documents, identifies gaps, generates quotes, and explains findings in plain English — all in a single conversation turn.

↓
🌐 React SPA

Filing cabinet, chat interface, and quote panel. Deployed on Cloudflare Pages with serverless Functions as API proxy.

🖥️ Claude Desktop

Alternative power-user interface via stdio↔HTTP bridge. Same MCP servers, richer tool visibility.
