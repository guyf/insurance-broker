Labeled Connections

From → To Credential / Flag Detail
Web Frontend Path (primary)
Browser → Cloudflare Pages — HTTPS · React SPA served as static assets
CF Pages /api/chat → Anthropic API ANTHROPIC_API_KEY POST /v1/messages · claude-sonnet-4-6 · agentic tool-use loop (up to 10 iterations)
CF Pages /api/chat → broker-mcp — HTTPS MCP tool calls: search_insurance_docs · list_policies · get_renewal_calendar
CF Pages /api/chat → quote-mcp — HTTPS MCP tool calls: get_home_quote · get_motor_quote · get_pet_quote · analyze_photo
CF Pages /api/policies → broker-mcp — HTTPS MCP · list_policies · populates the filing cabinet sidebar
CF Pages /api/upload → broker-mcp — HTTPS · multipart form · streams PDF to broker-mcp /upload endpoint for live ingestion
CF Pages /api/update-policy → broker-mcp — HTTPS PATCH · proxies inline card edits · broker-mcp calls update_policy_metadata() RPC
Claude Desktop Path (alternative)
Claude Desktop → supergateway — stdio (MCP protocol) — local process communication
supergateway → broker-mcp --streamableHttp HTTPS · streamable-http MCP · insurance-broker-production-85e3.up.railway.app/mcp
supergateway → quote-mcp --streamableHttp HTTPS · streamable-http MCP · alluring-prosperity-production-5644.up.railway.app/mcp
Shared Backend
broker-mcp → OpenAI OPENAI_API_KEY POST /v1/embeddings · model: text-embedding-3-small · query-time only
broker-mcp → Supabase SUPABASE_URL + SUPABASE_ANON_KEY / SERVICE_ROLE_KEY PostgREST RPC calls · search_documents · list_policies · get_renewal_calendar · update_policy_metadata
quote-mcp → OpenAI OPENAI_API_KEY POST /v1/chat/completions · model: gpt-4o-mini · analyze_photo tool only
Ingestion (one-off, local)
ingestion → OpenAI OPENAI_API_KEY POST /v1/embeddings · text-embedding-3-small · batched · run locally
ingestion → Supabase SUPABASE_URL + SUPABASE_SERVICE_ROLE_KEY Upsert into public.documents · bypasses RLS · dedup via chunk_hash
Google Drive → ingestion — Local CloudStorage mount · read-only · ~/Library/CloudStorage/GoogleDrive-…
CI / CD
GitHub → Cloudflare Pages CF_API_TOKEN (secret) GitHub Actions on push to main · builds /frontend · deploys static + functions
